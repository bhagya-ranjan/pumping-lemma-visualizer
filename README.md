## 🎯 Project Description

This project helps students understand the **Pumping Lemma** using:

* ✂️ String decomposition into **x, y, z**
* 🔁 Pumping (repetition of substring y)
* 📊 Testing multiple values of **k**
* ❌ Identifying non-regular languages using contradiction

---

## 🧠 Key Concept

The Pumping Lemma states:

> If a language is regular, then any sufficiently long string can be split into
> **s = xyz**, such that:
>
> * |xy| ≤ p
> * |y| > 0
> * xyᵏz ∈ L for all k ≥ 0

---

## ⚙️ Features

* 🎮 Interactive simulator
* 🔁 Automatic testing for multiple values of k (0–5)
* 📊 Visual result comparison (✔ / ✖)
* 🧠 Correct theoretical conclusion
* 🎨 Modern UI with Tailwind CSS

---

## 🚀 How It Works

1. Enter a string
2. Choose a language
3. Click **Simulate Split**
4. The system:

   * Splits string into x, y, z
   * Tests multiple values of k
   * Displays results
   * Gives final conclusion

---

## ⚠️ Important Note

* If **any value of k fails**, the language is **NOT regular** ❌
* If all tested values pass, the language **may be regular** (not guaranteed) ⚠️
