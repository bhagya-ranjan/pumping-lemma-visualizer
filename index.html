Pumping Lemma Theory

Definition

Any sufficiently long string can be split into x, y, z such that repeating y keeps the string valid.

|xy| ≤ p
|y| > 0
xykz ∈ L

String Breakdown

Pumping Process

k = 2 → aaabbbbccc
k = 3 → aaabbbbbbccc

Increasing k repeats y.

Intuition

Finite automata have limited memory. When processing long strings, they loop. That loop corresponds to y.
